# Open_Source_Audit
This project explores open-source software through Linux, analyzing its structure, licensing, and ecosystem. It includes shell scripts to audit systems, demonstrating practical skills and OSS concepts
Open Source Audit Project – 24BCE10518 Name: Priyanshi Solanki Roll No: 24BCE10518 Project: Open Source Audit Overview This project includes 5 shell scripts to show off your Linux skills and your understanding of opensource software and how it works. Scripts

System Identity Report (script1.sh) This script will show information such as operating system, kernel, username, and uptime.
FOSS Package Inspector (script2.sh) This script will check whether a package is installed or not and show information about it.
Disk & Permission Auditor (script3.sh) This script will show directory permissions and disk usage.
Log File Analyzer (script4.sh) This script will read log files, count the number of keywords, and show recent log entries.
Open Source Manifesto (script5.sh) This script will generate a simple open-source message using user input.
